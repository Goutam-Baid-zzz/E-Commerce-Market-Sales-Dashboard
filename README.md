# 📊 Indian E-Commerce Pricing & Revenue Analysis

An interactive, insight-driven **Power BI dashboard** designed to analyze pricing strategies, sales performance, and revenue trends across the Indian e-commerce market over a **36-month period**.

---

## 🚀 Project Overview

This project explores how **pricing, discounts, customer segments, and regional demand** influence revenue.  
Using advanced visualizations and drill-down tooltips, the dashboard converts raw transactional data into **actionable business intelligence**.

---

## 📌 Key Metrics

- 💰 **Total Revenue:** ₹2.17 Billion  
- 📦 **Total Units Sold:** 970K  
- 📅 **Time Period:** 36 Months  
- 🎯 **Goal Tracking:** Revenue vs Target comparison  

---

## 📊 Dashboard Capabilities

### 🔹 Main Dashboard (Executive View)

- 🌍 **Regional Analysis:** Revenue by zone (North, South, West, Central, East)  
- 📈 **Trend Analysis:** Monthly revenue patterns & seasonality  
- 👥 **Customer Insights:** Gender-based sales contribution  
- 🛍️ **Category Performance:** Revenue by product categories  
- 🏷️ **Brand Segmentation:** Mass vs Premium comparison  
- 🎉 **Event Analysis:** Festival vs Normal sales impact  
- 🎛️ **Interactive Filters:** Year, State, Category  

---

### 🔸 Tooltip Drilldowns (Deep Insights)

- 📍 **Revenue by State per Zone** → Regional contribution breakdown  
- 📋 **State Performance Table** → Units sold + revenue trends  
- 🏆 **Top 5 States** → Highest revenue contributors  

---

## 📁 Dataset Overview

This project uses a structured dataset simulating real-world e-commerce operations:

| Category | Features |
|--------|--------|
| **Pricing** | base_price, discount_percent, final_price |
| **Sales** | revenue, units_sold |
| **Customer** | age, gender |
| **Product** | category, brand_type |
| **Market Factors** | competition_intensity, inventory_pressure |
| **Geography** | state, zone |
| **Time** | order_date, order_year |

---

## 🛠️ Tech Stack

- **Power BI** → Dashboard & Visualization  
- **DAX** → Measures, KPIs, Time Intelligence  
- **Power Query** → Data Cleaning & Transformation  

---

## 📈 Key Business Insights

- 📊 **South & North zones** dominate total revenue  
- 💎 **Premium & Electronics categories** drive the highest value  
- 🎉 **Festival events significantly boost sales**  
- 📍 **Top States:** Delhi NCR, Maharashtra, Karnataka  
- 📉 Revenue shows **stable trends with seasonal spikes**
